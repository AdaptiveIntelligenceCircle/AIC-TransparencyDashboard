# Code of Conduct - AIC-TransparencyDashboard

We expect all contributors to act with respect, honesty, and integrity.

**Key Rules:**
- No harassment, discrimination, or personal attacks.
- Transparency is non-negotiable — do not propose features that reduce visibility.
- Respect third path absolute and zero-donation principles.
- Any violation may result in removal from the project.

This project follows the Contributor Covenant with the addition of AIC’s core principles.
