# Contributing Guidelines - AIC-TransparencyDashboard

Thank you for contributing to the official transparency layer of AIC/HMN.

### Core Principles (Non-Negotiable)
- Radical transparency is fundamental.
- All contributions must uphold **third path absolute** and **zero-donation**.
- The dashboard must remain fully auditable and public.

### How to Contribute
1. Fork the repository.
2. Create a feature branch.
3. Make your changes with clear documentation.
4. Open a Pull Request with detailed explanation.

**During the Founder’s military service (2027–2029):**  
Only maintenance, bug fixes, and documentation updates are accepted.

### What We Do NOT Accept
- Any feature that hides information or reduces transparency.
- Attempts to add centralized control or backdoors.
- Commercial or dual-licensing proposals.

By contributing, you agree to this file, the Code of Conduct, and all policies in `/POLICIES`.
